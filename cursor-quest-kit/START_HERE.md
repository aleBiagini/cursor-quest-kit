# Start Here — Prompt Unico

Copia questo prompt in una nuova chat Cursor aperta sulla root del tuo progetto:

```text
Usa la skill project-quest-runner in /cursor-quest-kit/.cursor/skills/project-quest-runner/SKILL.md.
Voglio un laboratorio pratico personalizzato per questo repository.

Vincoli:
- crea esattamente 3 quest progressive (small, medium, advanced);
- niente dipendenze MCP obbligatorie;
- includi criteri di accettazione verificabili e prompt eseguibile per ogni quest;
- non inventare file/comandi che non esistono nel repo.

Durata target: 60 minuti.
Livello: intermedio.
Output nel formato "Tailored Quest Pack" definito nella skill.
```

## Uso consigliato

1. Esegui prima la Quest 1.
2. Aggiorna `QUEST_STATE.md`.
3. Passa alla Quest successiva solo dopo aver chiuso i check della precedente.
